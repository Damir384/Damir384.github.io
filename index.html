Полезные интернет сайты

VirusTotal

На VirusTotal можно проверить файлы на
наличие вирусов с помощью многих
антивирусов.

Google

Google - это крупнейший поисковик по
интернету.

Firefox

Firefox - это один из многочисленых браузеров позволяющий посматривать интернет контент.

Duckduckgo

Duckduckgo - это анонимный поисковик
который не записывает историю поиска.

Wikipedia

Wikipedia - это общедоступная многоязычная универсальная интернет-энциклопедия со свободным контентом.

Github

Github - это крупнейший веб-сервис для
хранения IT-проектов и их совместной
разработки.
